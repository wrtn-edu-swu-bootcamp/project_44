# 서울여자대학교 <span style="background-color: #90EE90;">빈 강의실</span> 현황 실시간 알리미 기획안

## 1. 서비스 개요

### 1.1 서비스명
**서울여자대학교 <span style="background-color: #90EE90;">빈 강의실</span> 현황 실시간 알리미**

### 1.2 서비스 목적
서울여자대학교 재학생들이 실시간으로 <span style="background-color: #90EE90;">빈 강의실</span>을 확인하여 자습 공간을 쉽게 찾을 수 있도록 돕는 서비스입니다.

### 1.3 해결하고자 하는 문제
- 학생들이 자습할 공간을 찾기 위해 여러 건물과 강의실을 직접 돌아다니며 확인해야 하는 불편함
- 강의실이 사용 중인지 여부를 사전에 알 수 없어 시간 낭비가 발생하는 문제
- 특정 시간대에 <span style="background-color: #90EE90;">빈 강의실</span>이 어디인지 한눈에 파악하기 어려운 문제

### 1.4 서비스 가치
- **시간 절약**: 실시간으로 <span style="background-color: #90EE90;">빈 강의실</span>을 확인하여 불필요한 이동 최소화
- **편의성 향상**: 모바일이나 웹에서 언제 어디서나 강의실 현황 확인 가능
- **학습 환경 개선**: 적절한 자습 공간을 빠르게 찾아 학습 효율성 증대

## 2. 사용자 페르소나

### 2.1 주요 타겟
**서울여자대학교 재학생**

### 2.2 사용 목적
- 자습 공간 찾기
- 조별 과제나 스터디를 위한 공간 찾기

### 2.3 사용자 니즈

- 특정 시간대(예: 오후 2시~4시)에 <span style="background-color: #90EE90;">빈 강의실</span> 조회
- 강의실 위치(건물, 층수) 정보 확인
- 강의실 수용 인원 정보로 적합한 공간 선택

## 3. 핵심 기능

### 3.1 현재 시간 기준 <span style="background-color: #90EE90;">빈 강의실</span> 조회
- 사용자가 접속한 현재 시각을 기준으로 즉시 사용 가능한 <span style="background-color: #90EE90;">빈 강의실</span> 목록 제공
- 각 강의실의 다음 수업 시작 시간까지 남은 시간 표시
- 실시간으로 업데이트되는 강의실 현황

### 3.2 특정 시간대 <span style="background-color: #90EE90;">빈 강의실</span> 조회
- 사용자가 원하는 시간대를 선택하여 해당 시간에 <span style="background-color: #90EE90;">빈 강의실</span> 조회
- 예: "오후 2시~4시에 <span style="background-color: #90EE90;">빈 강의실</span>"
- 시간대별 강의실 사용 현황을 한눈에 파악 가능

### 3.3 고급 필터링 및 검색
- **건물별 필터링**: 특정 건물의 강의실만 조회
- **층수별 필터링**: 원하는 층의 강의실만 표시
- **수용 인원별 필터링**: 필요한 인원 수에 맞는 강의실만 조회
- **시설별 필터링**: 프로젝터, 화이트보드, 컴퓨터 등 특정 시설이 있는 강의실만 표시
- **접근성 필터링**: 휠체어 접근 가능, 엘리베이터 이용 가능 등 접근성 정보 기반 필터링
- **검색 기능**: 강의실 번호나 건물명으로 직접 검색

### 3.4 지도 및 플로어플랜 뷰
- **캠퍼스 지도**: 건물 위치를 지도에서 확인
- **건물 내부 플로어플랜**: 각 건물의 층별 강의실 배치도 제공
- **시각적 상태 표시**: 지도/플로어플랜에서 강의실 상태를 색상으로 표시
  - <span style="background-color: #90EE90;">녹색</span>: 사용 가능 (빈 강의실)
  - <span style="background-color: #FF6B6B;">빨간색</span>: 사용 중 (수업 진행 중)
  - <span style="background-color: #FFD93D;">노란색</span>: 곧 사용 예정 (10분 이내 수업 시작)

### 3.5 알림 기능
- **즉시 알림**: 관심 강의실이 비면 실시간 알림
- **예약 알림**: 특정 시간대에 빈 강의실이 생기면 알림
- **다음 수업 시작 알림**: 현재 사용 중인 강의실의 수업 종료 시간 알림
- **푸시 알림**: 모바일 앱을 통한 푸시 알림 지원

## 4. 강의실 정보 구조

각 강의실에 표시되는 정보는 다음과 같습니다:

### 4.1 기본 정보
- **건물명/위치**: 강의실이 위치한 건물명 (예: 본관, 인문관, 자연관 등)
- **강의실 번호**: 강의실 고유 번호 (예: 101호, 201호 등)

### 4.2 실시간 상태 정보
- **수업 진행 중 여부**: 현재 해당 강의실에서 수업이 진행 중인지 여부
- **현재 인원/수용 인원**: 강의실에 현재 있는 인원 수와 최대 수용 인원 수
  - 예: "15/30명" (현재 15명, 최대 30명 수용)

### 4.3 시간 정보
- **다음 수업 시작 시간**: 현재 <span style="background-color: #90EE90;">빈 강의실</span>인 경우, 다음 수업이 시작되는 시간
- **현재 수업 종료 시간**: 현재 수업이 진행 중인 경우, 수업이 끝나는 시간
- **남은 여유 시간**: 다음 수업까지 남은 시간 표시 (예: "1시간 30분 남음")
- **버퍼 타임**: 수업 간 정리 시간 고려 (기본 10분)

### 4.4 시설 정보
- **프로젝터**: 프로젝터 설치 여부
- **화이트보드/칠판**: 화이트보드 또는 칠판 유무
- **컴퓨터**: 강의실 내 컴퓨터 설치 여부
- **마이크/스피커**: 음향 시설 유무
- **에어컨/난방**: 온도 조절 시설
- **조명**: 조명 시설 상태

### 4.5 접근성 정보
- **휠체어 접근 가능**: 휠체어 사용자 접근 가능 여부
- **엘리베이터**: 엘리베이터 이용 가능 여부
- **장애인 화장실**: 인근 장애인 화장실 유무
- **시각 장애인 지원**: 점자 안내판, 음성 안내 등

### 4.6 강의실 상태 색상 표시
- **녹색 배경**: <span style="background-color: #90EE90;">빈 강의실</span> (현재 사용 가능)
- **빨간색 배경**: <span style="background-color: #FF6B6B; color: white;">수업 진행 중</span> (현재 사용 불가)
- **노란색 배경**: <span style="background-color: #FFD93D;">곧 사용 예정</span> (10분 이내 수업 시작)

- **실시간 업데이트**: 강의 일정 변경 시 즉시 반영

- **데이터 검증**: 다중 소스 데이터 교차 검증

### 7.2 운영 정책
- **버퍼 타임**: 수업 간 최소 10분 정리 시간 확보
- **노쇼 정책**: 예약 후 일정 시간 내 미체크인 시 자동 취소
- **사용 우선순위**: 
  1. 정규 수업 (최우선)
  2. 시험/평가
  3. 공식 행사
  4. 학생 자습/스터디
- **사용 시간 제한**: 연속 사용 시간 제한 (예: 4시간)
- **정리 의무**: 사용 후 강의실 정리 및 원상 복구

### 7.3 접근 제어
- **인증 시스템**: 서울여자대학교 포털 계정 연동 (SSO)
- **권한 관리**: 학생, 교수, 직원별 차등 권한
- **잠금 강의실**: 특별 관리 강의실 정보 제한 표시

   - 본관 101호: <span style="background-color: #90EE90;">빈 강의실</span> (다음 수업 16:00 시작)
   - 본관 201호: <span style="background-color: #90EE90;">빈 강의실</span> (다음 수업 없음)
   - 인문관 101호: <span style="background-color: #90EE90;">빈 강의실</span> (다음 수업 15:00 시작)
4. 학생 A가 본관 201호를 선택하여 이동
5. 강의실에서 자습 시작

### 8.2 시나리오 2: 특정 시간대 <span style="background-color: #90EE90;">빈 강의실</span> 조회

**상황**: 학생 B가 내일 오후 2시~4시에 조별 과제를 할 공간을 미리 찾고 있습니다.

**사용 과정**:
1. 학생 B가 서비스에 접속
2. "특정 시간대 조회" 메뉴 선택
3. 날짜: 내일, 시간대: 14:00~16:00 선택
4. 시스템이 해당 시간대에 <span style="background-color: #90EE90;">빈 강의실</span> 목록 표시
   - 본관 101호: <span style="background-color: #90EE90;">빈 강의실</span> (14:00~16:00 사용 가능)
   - 본관 201호: <span style="background-color: #90EE90;">빈 강의실</span> (14:00~16:00 사용 가능)
   - 인문관 101호: 14:00~15:00 사용 가능, 15:00 이후 수업 있음
5. 학생 B가 본관 201호를 선택하여 조별 과제 장소로 확정

### 8.3 시나리오 3: 수업 중인 강의실 확인

**상황**: 학생 C가 특정 강의실로 가려고 하는데, 현재 사용 중인지 확인하고 싶습니다.

**사용 과정**:
1. 학생 C가 서비스에 접속
2. "현재 시간 기준 <span style="background-color: #90EE90;">빈 강의실</span>" 메뉴 선택
3. 원하는 강의실(예: 본관 101호) 검색
4. 시스템이 현재 상태 표시:
   - 본관 101호: 수업 진행 중 ("컴퓨터 활용", 13:00~14:00)
   - 현재 인원: 25/30명
   - 수업 종료 시간: 14:00
5. 학생 C가 14:00 이후에 다시 확인하거나 다른 <span style="background-color: #90EE90;">빈 강의실</span> 선택

### 8.4 시나리오 4: 필터링을 활용한 강의실 찾기

**상황**: 학생 D가 5명이 함께 할 조별 과제를 위해 프로젝터가 있는 강의실을 찾고 있습니다.

**사용 과정**:
1. 학생 D가 서비스에 접속
2. 필터 옵션 선택:
   - 수용 인원: 5명 이상
   - 시설: 프로젝터 필수
   - 건물: 본관 또는 인문관
3. 시스템이 조건에 맞는 <span style="background-color: #90EE90;">빈 강의실</span> 목록 표시
   - 본관 201호: <span style="background-color: #90EE90;">빈 강의실</span> (수용 50명, 프로젝터 있음)
   - 인문관 301호: <span style="background-color: #90EE90;">빈 강의실</span> (수용 100명, 프로젝터 있음)
4. 학생 D가 본관 201호를 선택 (적정 크기)
5. 지도/플로어플랜에서 정확한 위치 확인
6. 강의실로 이동하여 조별 과제 진행

    ↓

    ↓

## 9. 향후 계획

### 9.1 실제 강의 데이터 연동
- **e-Class 시스템 연동**: 서울여자대학교 e-Class에서 개설 강의 정보 자동 수집
- **학기별 자동 업데이트**: 학기 시작 전 및 학기 중 정기 데이터 갱신
- **강의 시간표 데이터 파싱**: HTML/API를 통한 시간표 데이터 자동 파싱
- **데이터 정제**: 불일치 데이터 검증 및 정제 프로세스 구축
- **공지사항 연동**: 강의실 변경, 수업 취소 등 공지사항 자동 반영

### 9.2 고급 기능 추가
- **즐겨찾기 기능**: 자주 가는 강의실 즐겨찾기 및 빠른 접근
- **QR 코드 체크인**: 강의실 도착 시 QR 코드 스캔으로 실시간 상태 업데이트
- **실시간 점유 감지**: IoT 센서(모션 센서, 비콘)를 통한 실시간 점유 상태 확인
- **예약 시스템 연동**: 강의실 예약 시스템과 연동하여 비정규 사용 반영
- **사용률 통계**: 강의실별, 시간대별 사용률 분석 및 대시보드 제공
- **에너지 절약 연동**: 빈 강의실 감지 시 조명/에어컨 자동 제어 (스마트 캠퍼스 연동)

### 9.3 UI/UX 개선
- **모바일 앱 개발**: iOS 및 Android 네이티브 앱 개발
- **반응형 웹 디자인**: 다양한 디바이스에서 최적화된 화면 제공
- **다크 모드**: 눈의 피로를 줄이는 다크 모드 지원
- **접근성 향상**: 스크린 리더 지원, 키보드 네비게이션 등 접근성 기능 강화
- **다국어 지원**: 한국어, 영어 지원 (향후 확장 가능)

### 9.4 서비스 확장 가능성
- **다른 대학으로의 확장**: 타 대학에도 동일 서비스 제공
- **통합 플랫폼**: 도서관 좌석, 스터디룸 등 다른 공간 정보 통합
- **AI 기반 추천**: 사용자 패턴 분석을 통한 맞춤형 강의실 추천
- **커뮤니티 기능**: 강의실 리뷰, 사용 팁 공유 등 커뮤니티 기능 추가
- **관리자 대시보드**: 강의실 사용률, 통계, 최적화 제안 등 관리자용 기능

### 9.5 참고 사례 및 벤치마킹
- **StudySpace (Empty Room Finder)**: 개인 수업 일정 연동, 그룹 스터디 지원
- **UTD Rooms**: 공식 강의 일정 데이터 스크랩, 실시간 빈 강의실 표시
- **FIU Rooms Finder**: 시간대별 빈 강의실 조회, 남은 시간 표시
- **Availo**: 색상 기반 상태 표시 (녹색/빨강/노랑), 실시간 이용 상태
- **Smart Share**: 다양한 자원 통합 플랫폼, QR 코드 및 센서 연동
